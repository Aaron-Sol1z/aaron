Recipe Search

Search for dishes and find links to instructions and tutorial videos. This application pulls data from TheMealDB, which stores information about each dish.

When choosing this project, I wanted to make something that would incorporate a little bit of everything I've learned about react.js. I saw a similar project to this one and decided to make one of my own with additional features with the additional data retrieved from the API, such as rendering tags and the dish's nationality. While creating each component was straightforward, I ran into difficulty using react hooks and fetching data. Even so, I found my experience using react.js to be robust and fun.
