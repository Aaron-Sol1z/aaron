Dictionary

Search for a word and retrieve information such as definitions, example sentences, and phonetic spelling and pronunciations. This application pulls data from the Free Dictionary API, which stores information about dictionary entries.

While I had some experience working with APIs and appending DOM elements with JavaScript, I ran into challenges when retrieving data from a more complicated object. Since words have a varied amount of definition entries, I had the task to navigate through arrays to display every available information for each search result. I also ran into issues with playing a sound button for results that contained a phonetic spelling and audio file. Handling both issues required if statements and for loops to check if a search result returned certain information and print everything out to the screen.
