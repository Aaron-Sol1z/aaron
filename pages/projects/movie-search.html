Description

Search for details about a movie or series and read about them in a clean display. This application pulls data from the Open Movie Database (OMDb), which stores information about a movie or series.

When I was first learning HTML, CSS, and JavaScript, I had done a sample weather app project where I practiced fetching data from an API and rendering information with JavaScript. Between my new knowledge with front-end technologies and passion for entertainment, I was inspired to use the skills I learned to make a similar project akin to the likes of IMDb and Rotten Tomatos. Throughout this project, it was important to me that the fetched data was presented neatly, so work on the DOM elements and CSS files was also emphasized.
