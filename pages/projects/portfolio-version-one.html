Portfolio Version 1

This is the first version of my portfolio website. Like this one, it's main goal was to inform you about myself.

This portfolio site was a way for me to show my experience with front-end technologies as well and introduce myself to you. While scouring the Internet for examples of portfolios, my main takeaway was that without an about me section, a display of my work, and contact information, my portfolio page wouldn't be doing its job. So it was important for me to include all of those components on one website. While I find this version to be a bit busy compared to this one, I'm still glad to say that I have learned something from this. This is where I gained experience in CSS and page layout.
